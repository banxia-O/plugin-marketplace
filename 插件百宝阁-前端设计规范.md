# 插件百宝阁 — 前端设计规范

> 风格关键词：清透、柔和、亲切、轻量
> 目标用户：非技术人群（初中生/高中生也能无压力使用）
> 框架：React（推荐） | 托管：CF Pages

## 一、配色系统

### 核心色板

```css
:root {
  /* ── 背景层 ── */
  --bg-base:       #F6F2FF;       /* 极淡紫底色，全局背景 */
  --bg-surface:    #FFFFFF;       /* 卡片/内容区白底 */
  --bg-elevated:   #FAF8FF;       /* 悬浮层/导航栏底色 */

  /* ── 主色：淡紫 ── */
  --primary:       #9B8FFF;       /* 主按钮、Logo、选中态 */
  --primary-hover: #8578F0;       /* 主色悬浮 */
  --primary-light: #E8E4FF;       /* 主色浅底（标签背景、选中行） */
  --primary-subtle:#F3F0FF;       /* 极浅紫（区块背景提示） */

  /* ── 交互色：淡青 ── */
  --accent-cyan:       #5CC6BA;   /* 链接、交互按钮、次要操作 */
  --accent-cyan-hover: #49B3A7;
  --accent-cyan-light: #E0F5F1;   /* 青色浅底 */

  /* ── 强调色：淡橘 ── */
  --accent-warm:       #FF9F6C;   /* 热门标签、新上架徽章、通知 */
  --accent-warm-hover: #F08D5A;
  --accent-warm-light: #FFF0E8;   /* 橘色浅底 */

  /* ── 语义色 ── */
  --success:       #5DBE8A;       /* 审核通过 ✅ */
  --warning:       #F5C542;       /* 待完善 ⚠️ */
  --danger:        #E86B6B;       /* 未通过 ❌ / 错误 */

  /* ── 文字层级 ── */
  --text-primary:  #2D2A3E;       /* 主文本（深灰紫） */
  --text-secondary:#7B7890;       /* 次要文本 */
  --text-tertiary: #ABA8BE;       /* 辅助说明（仅装饰，不承载关键信息） */
  --text-inverse:  #FFFFFF;       /* 深色背景上的白色文字 */

  /* ── 边框与分割 ── */
  --border:        #E8E5F0;       /* 常规边框 */
  --border-light:  #F0EDF8;       /* 轻边框（卡片内分割线） */
  --divider:       #EDEAF5;       /* 区域分割线 */

  /* ── 阴影 ── */
  --shadow-sm:     0 1px 3px rgba(45, 42, 62, 0.06);
  --shadow-md:     0 4px 12px rgba(45, 42, 62, 0.08);
  --shadow-lg:     0 8px 24px rgba(45, 42, 62, 0.10);
  --shadow-hover:  0 8px 24px rgba(155, 143, 255, 0.15);   /* 悬浮时带主色调阴影 */
}
```

### 色彩使用规则

- **淡紫**占比最大：背景、主按钮、Logo、选中态、活跃标签
- **淡青**做交互引导：链接文字、次要按钮、图标 hover 态、进度条
- **淡橘**做重点吸引：热门徽章 🔥、新上架标记 🆕、下载量高亮、通知红点
- 三色不要同时出现在一个小区域内，保持视觉层次
- 所有颜色使用 CSS 变量，禁止硬编码 hex 值

## 二、字体

### 字体栈

```css
/* Google Fonts 引入 */
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@300;400;500;700&family=Plus+Jakarta+Sans:wght@400;500;600;700;800&family=JetBrains+Mono:wght@400;500&display=swap');

:root {
  --font-body:  'Plus Jakarta Sans', 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', sans-serif;
  --font-code:  'JetBrains Mono', 'Fira Code', monospace;
}

body {
  font-family: var(--font-body);
  font-size: 15px;
  line-height: 1.7;
  color: var(--text-primary);
  -webkit-font-smoothing: antialiased;
}
```

### 字号体系

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| Display | `clamp(28px, 5vw, 40px)` | 800 | 首页大标题"插件百宝阁" |
| H1 | `clamp(22px, 4vw, 28px)` | 700 | 分类页标题 |
| H2 | `clamp(18px, 3vw, 22px)` | 600 | 卡片内标题、区块标题 |
| H3 | `16px` | 600 | 小标题 |
| Body | `15px` | 400 | 正文 |
| Small | `13px` | 400 | 辅助信息、时间戳 |
| Code | `14px` | 400 | 代码/配置片段（用 `--font-code`） |

### 中文排版

```css
/* 中文行高比英文更大 */
p, li { line-height: 1.8; }

/* 中文标题轻微字间距 */
h1, h2, h3 { letter-spacing: 0.03em; }
```

## 八、动效

### 卡片入场（瀑布加载）

```css
@keyframes fadeUp {
  from { opacity: 0; transform: translateY(16px); }
  to   { opacity: 1; transform: translateY(0); }
}
.plugin-card {
  opacity: 0;
  animation: fadeUp 0.4s cubic-bezier(0.22, 1, 0.36, 1) forwards;
}
.plugin-card:nth-child(1) { animation-delay: 0s; }
.plugin-card:nth-child(2) { animation-delay: 0.06s; }
.plugin-card:nth-child(3) { animation-delay: 0.12s; }
.plugin-card:nth-child(4) { animation-delay: 0.18s; }
/* 第5张起不再延迟，避免等太久 */
.plugin-card:nth-child(n+5) { animation-delay: 0.24s; }
```

### 按钮弹簧

```css
.btn {
  transition: transform 0.25s cubic-bezier(0.34, 1.56, 0.64, 1),
              box-shadow 0.2s ease;
}
```

### 标签筛选切换

```css
.filter-tag {
  transition: all 0.15s ease;
}
.filter-tag.active {
  background: var(--primary);
  color: var(--text-inverse);
}
```

### 无障碍动效降级（必须加）

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
  }
}
```

## 九、响应式断点

```css
/* 移动优先 */
/* 默认 = 手机（<640px）*/
@media (min-width: 640px)  { /* 平板 */ }
@media (min-width: 1024px) { /* 桌面 */ }
@media (min-width: 1440px) { /* 大屏 */ }
```

关键适配：

- 移动端：单列卡片、搜索框全宽、分类标签横向滚动、导航栏汉堡菜单
- 平板：双列卡片
- 桌面：3-4 列卡片、侧边筛选（分类页）
- 移动端正文最小 14px，点击目标最小 44×44px
- 禁止横向滚动

## 十二、噪点纹理（可选，增加质感）

```css
body::after {
  content: '';
  position: fixed;
  inset: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='n'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23n)'/%3E%3C/svg%3E");
  opacity: 0.025;
  pointer-events: none;
  z-index: 9999;
}
```

极淡噪点，让纯色背景从"塑料感"变为"纸质感"。浅色场景 opacity 0.02-0.03 即可。
